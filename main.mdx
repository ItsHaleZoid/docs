## What is Asimov?

Asimov is a foundational AI search platform that enables developers to build powerful semantic search capabilities for AI agents and applications. It provides:

- **Vector-based semantic search** using state-of-the-art embeddings and re-rankings.
- **Content management** with automatic igestion, without hosting a sperate Vector Database.
- **Parameter filtering** using custom parameters
- **Usage tracking** of the the API Requests and Search events.
- **API-first architecture** for easy integration
