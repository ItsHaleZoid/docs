## What is Asimov?

Asimov is a foundational AI search platform that enables developers to build powerful semantic search capabilities for AI agents and applications. It provides:

- **Vector-based semantic search** using state-of-the-art embeddings
- **Content management** with automatic chunking and embedding generation
- **Metadata filtering** using custom parameters
- **Usage tracking** with tiered limits for free and paid users
- **API-first architecture** for easy integration
