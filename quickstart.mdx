Get up and running with Asimov in three simple steps.

## Step 1: Create an Account

1. Visit [Get Started](https://asimov.mov/get-started)
2. Sign up with your email address

## Step 2: Generate an API Key

1. Go to [API Keys](https://asimov.mov/settings/api)
2. Click the **"Generate"** button
3. Copy your API key (it starts with `asmv-`)



## Step 3: Make Your First Request

### Add Content

Add your first content to the knowledge base:

<CodeGroup>

```bash cURL
curl -X POST https://asimov.mov/api/sources/content/add \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "content": "Asimov is a foundational AI search platform that enables developers to build powerful search capabilities for AI agents and applications.",
    "name": "Introduction"
  }'
```

```javascript JavaScript
const response = await fetch('https://asimov.mov/api/sources/content/add', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer YOUR_API_KEY`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    content: 'Asimov is a foundational AI search platform that enables developers to build powerful search capabilities for AI agents and applications.',
    name: 'Introduction'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://asimov.mov/api/sources/content/add',
    headers={
        'Authorization': 'Bearer YOUR_API_KEY',
        'Content-Type': 'application/json'
    },
    json={
        'content': 'Asimov is a foundational AI search platform that enables developers to build powerful search capabilities for AI agents and applications.',
        'name': 'Introduction'
    }
)

print(response.json())
```

</CodeGroup>

### Search Content

Now search across your content:

<CodeGroup>

```bash cURL
curl -X POST https://asimov.mov/api/sources/content/search \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "What is Asimov?",
    "limit": 5
  }'
```

```javascript JavaScript
const response = await fetch('https://asimov.mov/api/sources/content/search', {

  },
  body: JSON.stringify({

const data = await response.json();
```
